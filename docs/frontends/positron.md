# Positron

The same extension as [VS Code](vscode.md); commands and settings listed on that page apply unchanged, including the bundled `scrutin` binary. If you prefer to point at your own install, follow the [install instructions](../getting-started.md#install) and set `scrutin.binaryPath` in settings.

![Positron](../assets/screenshot_positron_spell.png){ .screenshot }

## Installation

Install from the [Open VSX Registry](https://open-vsx.org/extension/VincentArel-Bundock/scrutin-runner), or clone the [GitHub repository](https://github.com/vincentarelbundock/scrutin) and build from source:

```bash
make positron
```
